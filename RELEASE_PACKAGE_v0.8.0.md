## 🚀 HOW TO USE

### First Time Setup
```
1. Download MilodikFX_v0.8.0.zip from Release page
2. Extract to folder (e.g., C:\Programs\MilodikFX)
3. Double-click MilodikFX.exe
4. Browser window opens automatically
5. Ready to use!
```

### Basic Workflow
```
1. Select audio device (left panel)
2. Click "+ ADD BLOCK" to add effects
3. Drag to reorder effects
4. Adjust parameters with knobs
5. Save preset with SAVE button
6. Switch scenes (1-4 buttons)
```

### Advanced Features
```
- Expression pedal mapping: EXP section
- Tuner: Click tuner icon in left panel
- Metronome: Tap tempo button
- Presets: LIBRARY tab
- Settings: SETTINGS tab
```

## 🆘 TROUBLESHOOTING

### Won't Start
- Check Windows version (10/11 64-bit required)
- Ensure resources folder is present
- Check firewall isn't blocking port 3000

### Browser Doesn't Open
- Set default browser (not Edge!)
- Manually open http://localhost:3000
- Check browser compatibility

### Performance Issues
- Close unnecessary apps
- Check CPU load graph
- Reduce effect count
- Increase buffer size
